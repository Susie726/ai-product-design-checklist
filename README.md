# AI Product Design Checklist

一份面向 AI 产品经理、Agent 产品设计者和垂类智能助手团队的实战型设计清单。

基于对《旅行助手类 AI 产品对比》的拆解与延展，提炼出适用于 **决策型 AI / 交易型 AI / 执行型 AI / Agent 产品** 的通用设计框架。

## 在线阅读

- 文章版：./analysis.md

## 这份内容适合谁

- AI 产品经理
- Agent / Copilot / 助手类产品设计者
- 交易型 AI / 垂类 AI 团队
- 想建立 AI 产品评审框架的人

## 核心结论

好的 AI 产品，不是“会聊天”，而是：

> 能持续理解用户约束，并把任务一步步推进到结果。
